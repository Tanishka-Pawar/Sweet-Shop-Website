# Sweet Shop Website

## Overview
The **Sweet Shop Website** is a **responsive front-end web application** designed to showcase a sweet shop’s products and offerings. It focuses on **clean design, intuitive navigation, and user-friendly layout**.

---

## Features
- Displays sweet shop products with categories and pricing.
- Responsive design compatible with **desktop and mobile devices**.
- Easy-to-navigate interface with a clean layout.
- Built entirely using **HTML, CSS, and JavaScript**.

---

## Technologies Used
- **HTML** – Structure and content of the website.
- **CSS** – Styling, layout, and responsive design.
- **JavaScript** – Interactivity and dynamic elements.
